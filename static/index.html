indonesian commute / transit schedules

Real-time schedule data for Indonesian transit.

Inamute provides structured access to commuter line schedules, station information, and route data. Built for developers building transit apps for Indonesian cities.

Status Early access. KRL Jabodetabek schedules available. More lines coming.
